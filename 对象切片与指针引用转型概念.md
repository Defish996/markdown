​	对于**static_cast**使用

int i = static_cast<int>(d); // 将 double 转为 int 他会将小数点后面的省略吗?

并不会：

```c++
    double d1 = 3.99;
    double d2 = -2.7;
    
    int i1 = static_cast<int>(d1); // 结果是 3
    int i2 = static_cast<int>(d2); // 结果是 -2

    cout << i1 << endl; // 输出: 3
    cout << i2 << endl; // 输出: -2
```

他会进行直接截断，并不进行四舍五入
